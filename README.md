# 格致

基于[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)搭建的个人博客。

* `Markdown`语法编辑文章
* 支持`Latex`公式（MathJex）
* 按年份归档文章
* 自动部署到`Github Page`

## 安装

本地调试需要安装以下第三方Python库

```bash
pip install mkdocs
pip install mkdocs-material
pip install pymdown-extensions
```
